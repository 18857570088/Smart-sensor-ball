# Smart sensor ball 用户使用说明

版本：1.0.0  
适用对象：Smart sensor ball Android APP 用户  
设备要求：支持蓝牙的 Android 手机、Smart sensor ball 智能拳击球设备

## 1. 产品简介

Smart sensor ball 是一款配合智能拳击球硬件使用的运动训练 APP。APP 通过蓝牙连接名称以 `SENBALL#` 开头的智能拳击球设备，接收设备电量和拳击计数数据，并提供训练计时、训练记录、成就、榜单和个人中心等功能。

当前版本的拳击次数来自智能拳击球蓝牙数据，不使用手机声音识别计数。

## 2. 首次使用

1. 安装并打开 `Smart sensor ball` APP。
2. 首次打开时，APP 会提示进入设置界面连接蓝牙设备。
3. 请确保智能拳击球设备已经开机，并靠近手机。
4. 根据系统提示允许蓝牙相关权限。部分 Android 系统可能要求定位权限才能进行蓝牙扫描，APP 不会主动获取您的位置轨迹。
5. 进入设置界面后，点击“扫描”，搜索 `SENBALL#` 开头的设备。
6. 选择设备后点击“连接”。
7. 连接成功后，主界面顶部蓝牙图标会变为蓝色，并显示电量信息。

## 3. 蓝牙连接

### 3.1 进入设置界面

在主界面点击右上角设置图标，进入“蓝牙连接及语言选择”界面。

### 3.2 扫描设备

点击“扫描”按钮后，APP 只显示名称以 `SENBALL#` 开头的设备。

- 如果只扫描到 1 个设备，APP 会自动选择该设备。
- 如果扫描到多个设备，请手动点击需要连接的设备。
- 未连接蓝牙时，“扫描”按钮可用。
- 选择设备后，“连接”按钮可用。

### 3.3 连接设备

选择设备后点击“连接”。连接成功后：

- 设置界面会显示已连接设备。
- “扫描”和“连接”按钮不可用。
- “断开”按钮可用。
- 主界面蓝牙图标显示为蓝色。
- 电池标识显示当前设备电量。

连接成功后，APP 会先发送关闭陀螺仪指令，保证设备在未训练时处于关闭计数状态。

### 3.4 断开设备

点击“断开”按钮后：

- 当前蓝牙连接会关闭。
- 设置界面恢复未连接状态。
- “扫描”按钮重新可用。
- 主界面蓝牙图标显示为红色。
- 电量显示为未连接状态。

## 4. 主界面说明

主界面顶部显示：

- APP 名称：`Smart sensor ball`
- 蓝牙状态图标：
  - 蓝色：已连接蓝牙设备
  - 红色：未连接蓝牙设备
- 电量标识：已连接设备后显示电量
- 设置图标：进入蓝牙连接和语言选择界面

底部主要模块包括：

- 锻炼中心
- 锻炼成果
- 榜单排名
- 个人中心

## 5. 开始训练

1. 确认蓝牙设备已连接。
2. 在“锻炼中心”选择训练模式。
3. 点击“开始”。
4. APP 会先显示并播放 `3、2、1、Go` 提示。
5. 提示结束后，训练倒计时正式开始。
6. 同时 APP 向设备发送开启陀螺仪指令，开始接收拳击计数。

训练期间，主界面会实时显示当前拳击次数。

## 6. 结束训练

训练可通过以下方式结束：

- 点击“结束”按钮。
- 训练倒计时结束。

训练结束后：

- APP 会向设备发送关闭陀螺仪指令。
- 停止接收本次训练计数。
- 生成训练结果。
- 保存训练记录。
- 根据成绩更新成就、榜单和个人数据。

## 7. 训练计数说明

当前版本的拳击次数来自智能拳击球蓝牙协议中的“数据2”。当设备识别到有效击球并使数据2增加时，APP 会同步增加拳击次数。

注意事项：

- 训练开始前，APP 会保持陀螺仪关闭。
- 点击“开始”并完成 `3、2、1、Go` 提示后，APP 才开启陀螺仪计数。
- 点击“结束”或训练时间结束后，APP 会关闭陀螺仪。
- 如果蓝牙断开，训练计数可能中断，请重新连接设备后再开始训练。

## 8. 锻炼成果

锻炼成果页面用于查看训练表现和成长记录。APP 会根据训练结果展示拳击次数、训练表现、连续训练情况、成就进度等信息。

部分数据需要网络连接才能同步到云端。

## 9. 榜单排名

榜单排名页面用于查看训练成绩排行。榜单可能包含：

- 30 秒最佳成绩
- 60 秒最佳成绩
- 累计拳击次数
- 连续训练表现

榜单数据依赖网络和云端服务。如果网络不可用，榜单可能无法刷新。

## 10. 个人中心

个人中心用于查看和管理个人资料及训练概览，包括：

- 昵称
- 头像
- 语言
- 训练统计
- 段位和成就摘要
- 隐私政策
- 用户协议

公司说明文字为：专注于智能拳击产品与运动数据体验。

## 11. 语言设置

进入设置界面后，可在“APP 语言”中选择界面语言。

当前支持：

- 简体中文
- English
- Français
- ไทย

语言设置会保存在当前设备中，后续打开 APP 时继续使用上次选择的语言。

## 12. 权限说明

APP 可能使用以下权限：

- 蓝牙权限：扫描、连接和读取智能拳击球数据。
- 定位相关权限：部分 Android 系统要求开启定位权限才能扫描蓝牙设备。
- 网络权限：同步训练记录、成就、榜单和个人资料。
- 存储或图片访问权限：用于选择或展示头像等本地图片。

当前版本不使用麦克风进行声音识别计数。

## 13. 隐私政策和用户协议

您可以在个人中心查看：

- 隐私政策
- 用户协议

使用 APP 前建议阅读相关内容。训练数据用于运动记录、统计、成就和榜单展示，不应作为医疗诊断或专业竞技判定依据。

## 14. 常见问题

### 14.1 扫描不到设备怎么办？

请按以下顺序检查：

1. 确认智能拳击球已经开机。
2. 确认设备名称以 `SENBALL#` 开头。
3. 确认手机蓝牙已开启。
4. 确认 APP 已获得蓝牙权限。
5. 部分 Android 系统还需要开启定位权限。
6. 将设备靠近手机后重新点击“扫描”。
7. 如果设备已被其他手机连接，请先断开其他连接后再扫描。

### 14.2 能扫描到设备但连接失败怎么办？

请尝试：

1. 点击“断开”，再重新连接。
2. 关闭手机蓝牙后重新打开。
3. 重启智能拳击球设备。
4. 重启 APP。
5. 确认设备电量充足。

### 14.3 连接成功但电量不显示怎么办？

电量来自蓝牙设备上报的数据。请确认：

1. 设备已经连接成功。
2. 设备已正常上报蓝牙数据。
3. 设备电量数据可能需要短暂等待后显示。
4. 如长时间不显示，请断开后重新连接。

### 14.4 点击“开始”后为什么不是立即计数？

APP 会先进行 `3、2、1、Go` 提示。提示结束后才开始倒计时，并向设备发送开启陀螺仪指令，此后才开始统计本次训练拳击次数。

### 14.5 点击“结束”后设备是否还会继续计数？

点击“结束”或训练时间结束时，APP 会发送关闭陀螺仪指令。正常情况下，设备会停止本次训练计数。

### 14.6 为什么训练记录或榜单没有刷新？

训练记录和榜单需要网络及云端服务支持。请检查手机网络，稍后刷新页面。

## 15. 安全提示

1. 使用前请确认周围空间安全，避免击打时碰撞他人或物品。
2. 请根据自身身体状况选择训练强度。
3. 未成年人应在监护人指导下使用。
4. 如果训练中出现不适，请立即停止训练。
5. 请勿在湿滑、拥挤或光线不足的环境中进行高强度训练。
