# Smart sensor ball

This repository contains the Smart sensor ball project source code.

## Directories

- `sensorball-android`: Android app source code.
- `sensorball-ios`: iOS app source code and Codemagic build workflow.
- `sensorball-server`: backend service source code.
- `sensorball-deploy`: deployment examples and database bootstrap files.
- `sensorball-docs`: project notes and deployment planning documents.
- `APPLOGO`: source logo assets.

Generated APK/IPA files, local build caches, signing files, server-password documents,
and backup folders are intentionally excluded from this repository.
